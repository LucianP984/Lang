
# Custom Programming Language Interpreter

This is a custom programming language interpreter implemented in Python. The language supports various features including arithmetic operations, boolean logic, string manipulation, control flow statements, classes, and more.

## Language Features

### Basic Arithmetic
- Addition, subtraction, multiplication, division
- Modulo operation
- Exponentiation (using `^` operator)
- Proper operator precedence

### Boolean Logic
- Boolean values (`true`, `false`)
- Logical operators (`and`, `or`, `!`)
- Comparison operators (`>`, `<`, `>=`, `<=`, `==`, `!=`)

### Text Manipulation
- String literals with double quotes
- String concatenation with `+`
- String operations

### Variables
- Global variable declaration and assignment
- Variable scoping

### Control Flow
- If-else statements
- While loops
- For loops (iteration over collections)
- Nested control structures

### Functions
- Function declaration and calling
- Return values
- Local variables
- Recursive functions

### Data Structures
- Lists with indexing
- Dictionaries with key-value pairs

### Object-Oriented Programming
- Class definitions
- Object instantiation with `new` keyword
- Methods and properties
- `this` keyword for self-reference

## Usage

To run a program written in this language:

python main.py <directory/source_file>
